# Conjure Kit

Allows users of [conjure.so](https://conjure.so) to connect and automize measurement synchronization between conjure.so and:

- [Trakt.tv](https://trakt.tv)
- [Rescue Time](https://rescuetime.com)
- more...

## Stack

- JS (React via CRA)
- Flowbite (UI)
